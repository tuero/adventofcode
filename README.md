# adventofcode
Solutions to the adventofcode.

# Instructions
To install,
```shell
# Clone
$ git clone https://github.com/tuero/adventofcode.git

# Enter the repository
$ cd adventofcode

# Compile
$ mkdir build && cd build
$ cmake ..
$ make
$ cd ..

# Running an example
$ cd bin/2020
$ ./2020_day1 < ../../data/2020/day1.txt
```
